# MiniGrid KeyCorridor S6R3

This file is scenario-author reference text. Generated workspaces receive
`task.md` and `task_contract.json`.

Use the 7x7 egocentric MiniGrid observation, direction, and mission text to
explore rooms, handle doors and keys, and pick up the requested object. Return an
integer action: `0` turn left, `1` turn right, `2` move forward, `3` pick up,
`4` drop, `5` toggle, or `6` done.
